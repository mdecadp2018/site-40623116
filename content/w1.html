w1

git clone  ：從遠端倉儲複製資料到指定資料夾

git add .   ：提交的所有修改放到暫存區

git comit -m： 暫存區的資料提交到分支

git push ：   將資料推送到遠端

cd  資料夾名稱：轉換當前的目錄位置

git checkout：替換工作區的版本

git status ：展示目前狀態

git branch 分支名 ：創建新分支

git log --oneline --graph --all ：檢查各提交間的關聯

git checkout 分支名 ：轉換分支

 git merge 分支名： 合併分支

git log ：查看版次
